Getting started
---------------

Clone or download this repository and install its dependencies:

```bash
git clone https://github.com/samme/phaser-ce-rollup.git my-phaser-game
# OR
npx degit "samme/phaser-ce-rollup" my-phaser-game

cd my-phaser-game
npm install

# Watch and serve
npm run dev

# Build for production
npm run build
```
